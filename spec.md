# Docker Environment Build Plan

## Project Overview
Build a multi-architecture Docker environment supporting ARM64 NVIDIA GPU deployments (CUDA) and M1 Apple Silicon deployments (MPS), with Python 3.12, PyTorch, and HuggingFace transformers using uv package manager.

## Requirements Summary

### Core Dependencies
- **Python**: 3.12
- **Package Manager**: uv
- **Configuration**: TOML file (pyproject.toml or uv.toml)
- **GPU Support**: CUDA (NVIDIA ARM64) + MPS (Apple Silicon M1)

### Architecture Support
- **Target 1**: ARM64 servers with NVIDIA GPUs
  - Architecture value: `linux/arm64`
  - GPU: CUDA 11.8/12.x
  - Hardware: x86_64 host, ARM64 container with CUDA support
- **Target 2**: Apple Silicon M1 microchips
  - Architecture value: `mac-aarch64`
  - GPU: MPS (Metal Performance Shaders)
  - Hardware: M1 Macs

## Implementation Approach

### 1. Project Structure
```
dscontainer/
├── Dockerfile              # Multi-architecture Docker file with ARG arch support
├── pyproject.toml          # uv configuration and Python dependencies
├── uv.lock                 # Dependency lock file (generated by uv)
├── .dockerignore           # Docker build exclusions
├── README.md               # Usage instructions

```
